# 📈 SOT Market Intelligence Dashboard

A professional real-time market intelligence dashboard built by Samuel Oyedokun.

## Features
- 📊 Live candlestick charts with technical indicators (MA20/50, Bollinger Bands, RSI, Volume)
- 💼 Portfolio tracker with P&L, allocation pie chart, and performance metrics
- 📰 News sentiment analysis for any selected asset
- 🏆 Market overview with performance comparison across US Stocks, Crypto, and NGX
- 🔔 Price alert system (high/low notifications)

## Tech Stack
- Python · Streamlit · Plotly · yFinance · Pandas · NumPy
